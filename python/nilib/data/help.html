HELP for Nilib Python Server

The server responds to the following at ==SERVER_NAME==:

GET requests on paths:

  • /help.html,
  • /testapp, (prints 'Hello World!'),
  • /envapp, (outputs WSGI environment dictionary),
  • /.well-known/ni[h]/<digest algorithm id>/<digest>,
  • /ni_cache/<digest algorithm id>;<digest>,
  • /ni_meta/<digest algorithm id>;<digest>,
  • /getputform.html,
  • /nrsconfig.html, (when running NRS server)
  • /favicon.ico,
  • /netinfproto/list, and
  • /netinfproto/checkcache

POST via the getputform.html form on paths (basic system):

  • /netinfproto/get,
  • /netinfproto/publish,
  • /netinfproto/put, and
  • /netinf/search

POST via the nrsconfig.html form on paths (when running NRS server):

  • /netinfproto/nrsconf,
  • /netinfproto/nrslookup,
  • /netinfproto/nrsdelete, and
  • /netinfproto/nrsvals

Documentation:

Documentation for the Python nilib code can be accessed at /doc.
